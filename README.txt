== Description ==
Many studies show that the abandoned cart rate can be as high as 60-70% which is certainly a major source of concern for e-commerce firms. There are multiple factors behind an abandoned cart but there are only a few proven steps available to reduce or recover such abandoned carts. One of these proven steps is sending follow-up emails which help to recover more than 30% of sales that might otherwise be lost. Zeta Abandoned Cart has all you need to make this process seamless and effective. It automates the whole follow-up email process. Whatâ€™s more, you can add automated coupons to further improve the recovery rate. Zeta Abandoned Cart has easy to use interface, the flexibility to set an interval when to consider a cart abandoned, the interval for sending follow-up emails, & it has a simple yet powerful dashboard to track the overall progress & analytics.

== Installation ==

There are two different ways to install the Zeta Abandoned Cart.

1. WP Plugins Directory
1. Manual Upload

**WP Plugins Directory**

1. Go to your **Wordpress Admin area > Plugins > Add New**.
1. Search for **Zeta Abandoned Cart**.
1. Install & Activate  **Zeta Abandoned Cart**.
1. Go to **Abandoned Cart > Settings** from your Wordpress Admin area, review & save your personalized settings.

**Manual Upload**

1. Go to your **WordPress Admin area > Plugins > Add New**.
1. Click on **Upload Plugin**.
1. Select & Submit the downloaded installable .zip file.
1. Activate plugin on successful installation.
1. Go to **Abandoned Cart > Settings** from your Wordpress Admin area, review & save your personalized settings.

== Frequently Asked Questions ==

= How Zeta Abandoned Cart collects customer information =

Zeta Abandoned Cart plugin carputers customer & cart information as soon as he/she fill in his/her correct email on checkout page.

= How to create Personalized coupon code =
Zeta Abandoned Cart plugin provides flexible option to create personalized coupon code automatically.

= How you mark a cart abandoned =

There's option to set time interval when system will consider a cart abandoned. Default time is 15 minutes.

= How to send follow up email =
Zeta Abandoned Cart plugin sends follow up email automatically on specified time interval.
You just need to setup email template & specify the time interval. Default interval is 10 hours.

== Features ==

* Captures detailed cart & trackable information of the customer seamlessly. 
* Set the time after which a cart is considered abandoned. 
* Elegant dashboard with recovery statistics & graphs. 
* Automatically sends notifications to admin (or specified email id) immediately when a cart gets abandoned. 
* Automatically sends a follow-up email within a specified time frame after the cart gets abandoned. 
* Copy HTML from anywhere & create a personalized follow-up email template using the powerful Rich Text Editor & utilizing variables such as name, direct cart links, etc. 
* Automatic Coupon creation for specific cart based on pre-defined criteria.  
* GDPR Integration
